## 示例代码

如需要运行示例项目，请 `clone` 当前 `repo` 到本地，然后从根目录下执行 `JSSwifter.xcodeproj`，打开项目后切换 `Scheme` 至 `JSSwifter-Tests`，`CMD+U` 即可运行测试示例代码。

## Swift 版本依赖
| Swift | JSSwifter | 
| ------| ----------|
| 5.0   | >= 0.0.1  |

## 限制条件
* **iOS 9.0** and Up
* **Xcode 10.0** and Up
* **Swift Version = 5.0**

## 安装

`JSSwifter` 可以通过 [CocoaPods](https://cocoapods.org) 获得。安装只需要在你项目的 `Podfile` 中添加如下字段：

```ruby
pod 'JSSwifter', '~> 0.0.9'
```
