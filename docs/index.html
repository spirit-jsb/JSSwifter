文档

Swift 版本依赖

Swift JSSwifter
5.0 >= 0.0.1

限制条件

  • iOS 9.0 and Up
  • Xcode 10.0 and Up
  • Swift Version = 5.0

安装

JSSwifter 可以通过 CocoaPods 获得。安装只需要在你项目的 Podfile 中添加如下字段：

pod 'JSSwifter', '~> 1.0.0'
